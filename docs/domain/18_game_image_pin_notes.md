# Game Image Pin

**Spielbeschreibung:** Der Spieler erhält eine Frage und muss den leuchtenden Ball als Pin an der richtigen Stelle im Bild platzieren. Wird der Pin korrekt gesetzt, bleibt das Bild intakt und die Aufgabe gilt als erfolgreich gelöst. Bei einer falschen Platzierung zerbricht das Bild, und die Runde muss erneut begonnen werden. Für jede korrekte Positionierung werden Punkte vergeben; die volle Punktzahl erhält der Spieler jedoch nur, wenn die richtige Lösung bereits im ersten Versuch erreicht wird

**Mechanics:** Im Zentrum der Spielmechanik steht die Beantwortung einer Frage durch die korrekte räumliche Platzierung eines Pins innerhalb eines Bildes. Jede Runde besteht aus einer Aufgabenstellung, einem Platzierungsversuch und einer unmittelbaren Rückmeldung auf die Aktion des Spielers. Eine korrekte Positionierung stabilisiert das Bild und belohnt den Spieler mit Punkten, während eine fehlerhafte Positionierung zum Scheitern der Runde führt und einen neuen Versuch auslöst. Die Punktevergabe ist leistungsabhängig gestaltet, da nur beim ersten korrekten Versuch die maximale Punktzahl erreicht werden kann; dies entspricht einer klaren Reward-Struktur innerhalb der Mechanics

**Dynamics:** Durch die Konsequenz, dass bei jeder falschen Platzierung das Bild zerbricht und die Runde neu gestartet wird, entsteht ein spürbarer Handlungsdruck. Diese Dynamik erzeugt Konzentration, eine erhöhte Fehlerbewusstheit und die Notwendigkeit, die eigene Entscheidung kritisch zu überprüfen. Gleichzeitig wird der Spieler dazu angehalten, seine Zuordnung reflektiert zu verbessern, bevor er einen erneuten Versuch unternimmt; allerdings sollte diese Form der Fehlerkonsequenz didaktisch ausgewogen bleiben, da Wiederholung idealerweise mit lernförderlichem Feedback verbunden wird

**Aesthetics:** Die angestrebte ästhetische Erfahrung liegt primär im Bereich der **Challenge** Das Spiel soll Spannung erzeugen, indem präzises Handeln, Aufmerksamkeit und korrektes Wissen unter dem Risiko eines unmittelbaren Scheiterns gefordert werden. Dadurch entsteht ein Spielerlebnis, das von Herausforderung, Zielorientierung und direktem Feedback geprägt ist
